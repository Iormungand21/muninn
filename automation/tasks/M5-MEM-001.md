# M5-MEM-001

## Goal
Implement episodic-to-semantic memory consolidation workflow.

## Scope
- Add `src/memory/consolidation.zig` with:
  - `findConsolidationCandidates()`: query episodic records older than a threshold (default: 7 days) that are not yet marked consolidated.
  - `consolidateCluster()`: given a list of related episodic records, produce a single semantic summary record. Use a simple heuristic (shared keys, overlapping content) — no LLM call needed for v1.
  - `markConsolidated()`: update source episodic records with `consolidated = true`.
  - `runConsolidation()`: orchestrate find → cluster → consolidate → mark.
- Wire `runConsolidation()` into the hygiene sweep (optional, controlled by config flag `memory.consolidation_enabled`, default false).
- Export from `src/memory/root.zig`.
- Add tests with mock episodic records verifying clustering and consolidation output.

## Acceptance
- Consolidation candidates identified by age and status
- Clustering produces semantic records from episodic clusters
- Source records marked as consolidated
- Wired into hygiene (disabled by default)
- Tests cover candidate finding, clustering, and marking
- All existing tests still pass
