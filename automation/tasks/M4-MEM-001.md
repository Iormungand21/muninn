# M4-MEM-001

## Goal
Add SQLite schema migration for TypedRecord metadata columns.

## Scope
- In `src/memory/sqlite.zig`:
  - Add migration step to add columns: `kind TEXT DEFAULT 'raw'`, `tier TEXT DEFAULT 'short_term'`, `source_channel TEXT`, `source_author TEXT`, `confidence REAL DEFAULT 1.0`.
  - Use `ALTER TABLE memories ADD COLUMN` with `IF NOT EXISTS` or catch the error for idempotent migration.
  - Update `store()` to write kind, tier, source, confidence when provided.
  - Update `recall()` to read and populate these fields in returned entries.
  - Backfill: existing rows get defaults (raw, short_term, 1.0 confidence).
- Add tests verifying migration is idempotent and new fields are persisted.

## Acceptance
- Migration adds new columns without data loss
- Migration is idempotent (safe to run multiple times)
- Store writes new fields, recall reads them
- Existing records work with defaults
- All existing memory tests still pass
