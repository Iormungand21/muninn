# M5-SEC-001

## Goal
Add audit log querying CLI command.

## Scope
- In `src/main.zig`, add an `audit` subcommand with actions:
  - `audit search --actor X --action Y --since 7d` — search audit logs by actor, action, and time range.
  - `audit tail -n 20` — show last N audit events.
  - `audit stats` — show counts by action type and actor.
- Add `src/security/audit_query.zig` (or extend `audit.zig`) with:
  - `searchEvents()`: parse JSONL audit log, filter by actor/action/time range.
  - `tailEvents()`: read last N lines from audit log file.
  - `computeAuditStats()`: count events by action and actor.
- Add tests for filtering and stats computation.

## Acceptance
- `nullclaw audit search` filters by actor/action/time
- `nullclaw audit tail` shows recent events
- `nullclaw audit stats` shows event counts
- Tests cover filtering and edge cases (empty log, no matches)
- All existing tests still pass
